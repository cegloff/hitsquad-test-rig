# Hitsquad smoke plan

Build a tiny Go CLI with three features that exercise the planner and
the dependency graph.

## Requirements

1. Add a `main.go` that defines a Cobra-style root command called
   `hellotool` with a version flag. When run without subcommands, it
   prints `hellotool v0.1.0` and exits zero.

2. Add a `greet` subcommand that takes a `--name` flag (default
   `world`) and prints `Hello, <name>!`. This depends on the root
   command existing first.

3. Add a README describing both commands with usage examples. This is
   documentation only and can run in parallel with (1) and (2).

Every change should compile, pass `go vet`, and include at least one
test asserting the new behavior.
